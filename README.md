# Description

Open [example](https://pet-custom-qr.web.app/)

## Installation

1.  `npm install` || `yarn`
1.  fill `.env.development` for dev or `.env.production` for prod by `env.example`

## Running the app:

### `NPM`

#### Development

Start dev server:

```sh
$ npm run start
```

### `YARN`

#### Development

Start dev server:

```sh
$ yarn start
```
